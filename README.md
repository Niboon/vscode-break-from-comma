# break-from-comma README

This is a string manipulation tool for people who don't like very long comma separated list.

## Features

Turn this:
```
const { very, long, comma, separated, values } = object
```
Into this:
```
const { 
    very, 
    long, 
    comma, 
    separated, 
    values } = object
```
